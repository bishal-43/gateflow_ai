# gateflow-ai


AI-powered smart event access and guest intelligence platform.


GateFlow AI helps organizers manage guest entry, QR verification,
walk-in approvals, AI-based guest assistance, and live attendance monitoring.


## Features

- QR-based guest verification
- Walk-in approval system
- AI guest assistant
- Voice AI caller assistant
- Real-time attendance dashboard
- Smart event monitoring


## Tech Stack

Frontend: React + tailwind
Backend: FastAPI
Database: PostgreSQL
AI: OpenAI / Gemini
Vector DB: pgvector
DevOps: Docker, Nginx
Cloud: AWS



## Structure

- frontend/ → React frontend
- backend/ → FastAPI backend
- ai-service/ → AI services
- docker/ → Docker configs
- nginx/ → Reverse proxy configs


## Team Roles

Frontend Developer -> UI development
Backend Developer  -> APIs and database
AI Developer       -> AI assistant and voice AI
DevOps             -> Docker, deployment, CI/CD
